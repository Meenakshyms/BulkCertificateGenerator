# Bulk Certificate Generator

An automated system to generate certificates in bulk using a predefined template and participant data.

## Features
- Generates certificates for multiple users at once
- Uses CSV input for participant details
- Reduces manual effort and errors

## Tech Stack
- Python
- Pandas

## How to Run
1. Place participant data in `data.csv`
2. Keep certificate template in project folder
3. Run `python app.py`
4. Generated certificates will be saved automatically
